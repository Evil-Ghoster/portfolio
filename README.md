## 🚀 Déploiement sur GitHub Pages

### Étape 1: Créer un repository GitHub
1. Allez sur [github.com](https://github.com) et créez un nouveau repository
2. Nommez-le: `Evil-Ghoster.github.io` (remplacez par votre username)
3. Laissez-le public

### Étape 2: Cloner et pousser votre code
```bash
# Initialiser le repository local
git init
git add .
git commit -m "Initial commit: Portfolio setup"

# Ajouter l'URL distante
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git

# Pousser vers main branch
git branch -M main
git push -u origin main
```

### Étape 3: Activer GitHub Pages
1. Allez dans **Settings** du repository
2. Allez à **Pages**
3. Sous "Build and deployment", sélectionnez:
   - Source: `GitHub Actions`
4. C'est tout ! 🎉

### Étape 4: Vérifier le déploiement
- Allez à l'onglet **Actions**
- Regardez le workflow `Deploy to GitHub Pages`
- Une fois ✅ réussi, votre site est en ligne !

## 📝 Apporter des modifications

À chaque modification de votre code:
```bash
git add .
git commit -m "Update: description de vos changements"
git push origin main
```

Le site se déploiera automatiquement en quelques secondes! ⚡
